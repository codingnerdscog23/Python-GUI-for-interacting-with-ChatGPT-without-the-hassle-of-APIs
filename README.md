# Python ChatGPT GUI Application

This Python application provides a user-friendly graphical interface to interact with ChatGPT. It utilizes a JSON file for session information and allows users to enter prompts and receive responses from ChatGPT without the need for direct API interaction.

## Features

- Load and use JSON file for ChatGPT session data
- Input field for text prompts to ChatGPT
- Configurable background image
- Start interaction with a Start button
- Safe exit with confirmation using an Exit button

## Prerequisites

- Python 3.x
- Tkinter (usually comes pre-installed with Python)
- PIL (if you want to use image formats other than PNG)
